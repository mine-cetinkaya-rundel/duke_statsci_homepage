2. Alternatively, use the following lines of code to load the data in R using the following:

<div class="highlight">

	bike <- read.csv("https://stat.duke.edu/~mc301/data/nc_bike_crash.csv", 
	sep = ";", stringsAsFactors = FALSE, na.strings = c("NA", "", "."))    
    </code>
